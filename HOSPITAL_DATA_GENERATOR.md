# 🏥 병원 데이터 생성기 V2

## 📋 개요

뇌질환케어 웹사이트의 병원 검색 기능을 위한 대규모 데모 데이터 생성 도구입니다.

```javascript
// "뇌혈관" 검색
currentFilters.search = '뇌혈관';
filterHospitals();

filterHospitals();
// 결과: ~500개 (서울 지역 병원)
```

### **2. 필터 기능**

```javascript
// 상급종합병원 필터
currentFilters.category = '상급종합';
filterHospitals();
// 결과: 정확히 500개

// 서울 + 전문병원 복합 필터
currentFilters.region = '서울';
currentFilters.category = '전문병원';
filterHospitals();
// 결과: 서울 전문병원만
```
